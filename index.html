Lenguajes y Tecnologías

Frontend

HTML

Lenguaje base para crear páginas web. Define la estructura y el contenido.

CSS

Se utiliza para diseñar y dar estilo a las páginas web.

JavaScript

Lenguaje de programación que se ejecuta en el navegador del usuario y permite crear páginas web interactivas y dinámicas.

Vue

Es un framework progresivo de JavaScript utilizado para construir interfaces de usuario dinámicas y reactivas.

Backend

Node.js

Es un entorno de ejecución que permite usar JavaScript en el servidor, se maneja solicitudes de usuarios, interacción con bases de datos y realizar operaciones en el servidor.

MongoDB

Base de datos NoSQL que almacena datos en formato de documentos JSON. Es ideal para aplicaciones que requieren manejar grandes volúmenes de datos no estructurados y cambiar rápidamente.

MySQL

Es un sistema de gestión de bases de datos relacional que organiza los datos en tablas. Es muy utilizado en aplicaciones web para almacenar y gestionar información estructurada.

Python

Lenguaje de programación muy versátil y fácil de aprender. Se utiliza en una amplia variedad de aplicaciones, desde desarrollo web hasta análisis de datos y aprendizaje automático.
